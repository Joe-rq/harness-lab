## 功能摘要

### `/harness-setup` Skill

在 Claude Code 环境中一键接入治理框架：
- 交互式选择安装模块
- 自动检测冲突文件
- 可选配置 PreToolUse hook
- 生成接入报告

### `harness:install` CLI

命令行安装脚本，支持：
- `--defaults` - 使用默认选项
- `--core-only` - 仅安装核心模块
- `--with-hook` - 包含 PreToolUse hook
- `--source <path>` - 指定源目录

## 使用方式

### Claude Code 环境

```
/harness-setup
```

### 命令行

```bash
# 交互模式
npm run harness:install

# 默认选项
npm run harness:install -- --defaults

# 带 hook
npm run harness:install -- --defaults --with-hook
```
