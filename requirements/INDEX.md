# REQ 索引

本目录存放所有 REQ（需求规格说明书）。

## 命名规则

```
requirements/{status}/REQ-{YYYY}-{NNN}-{brief-desc}.md
```

- `status`: `in-progress` | `completed` | `on-hold`
- `YYYY`: 年份
- `NNN`: 当年序号（001 开始）
- `brief-desc`: 简短英文描述（kebab-case）
