## 🚀 快速创建

需要创建技术文档时，使用以下模板：

### 架构设计模板
```markdown
# [模块/系统] 架构设计

## 概述
简要描述架构设计

## 架构图
[附架构图]

## 模块划分
- 模块 A：职责
- 模块 B：职责

## 接口定义
关键接口说明

## 数据流
数据如何流转

## 依赖关系
外部依赖说明

## 风险评估
潜在风险和缓解方案
```

### 规范文档模板
```markdown
# [类型] 规范

## 目的
为什么要这个规范

## 适用范围
适用于什么场景

## 具体规范
1. 规范 1
2. 规范 2

## 示例
好的示例和坏的示例

## 检查工具
如何自动检查
```
